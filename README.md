# Prueba-juego

Juego tipo Pong listo para jugar al descargar el repositorio.

## Cómo ejecutarlo en Visual Studio Code

1. Abre la carpeta del proyecto.
2. Abre el archivo `/tmp/workspace/YisusMula/Prueba-juego/index.html`.
3. Ejecuta con la extensión **Live Server** o abre el archivo en tu navegador.

## Funcionalidades

- Modo **1 jugador vs máquina**
- Modo **2 jugadores (jugador contra jugador)**
- **Marcador** para ambos lados
- **Selector de dificultad** para la IA (fácil, media, difícil)

## Controles

- Jugador 1: `W` / `S`
- Jugador 2 (modo 2 jugadores): `↑` / `↓`
